# Changelog

## W.I.P.
* Added publishing data for all entities once after discovery (to avoid unavailable/unknown states)
* Added support for periferals (addon sensor)
  - never disabled by default,
  - never falls into diagnostic category
  - enforces 2 decimals precision in HA
* Added support for `humidity` and `voltmeter` components
* added support for `input` (as binary sensor)
* renamed `ignore_names` into `custom_names`
* Docs Updated
